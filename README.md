# Circuit Theory & Network Analysis

This section covers the experimental validation of core circuit theorems, bridging the gap between nodal/mesh analysis theory and physical breadboard implementation.

## Key Experiments & Verifications
* **Thevenin & Norton Equivalents:** Reduced complex linear networks into simplified single-source equivalent circuits to analyze load interactions.
* **Superposition Theorem:** Verified the independent effect of multiple sources on specific branch currents and node voltages.
* **Maximum Power Transfer:** Experimentally determined the optimal load resistance ($R_L = R_{th}$) to achieve peak power efficiency in a given network.
* **Linearity & KCL/KVL:** Confirmed Kirchhoff’s laws through precise multimeter measurements and theoretical calculations.

## Lab Equipment
* Breadboard and passive components (Resistors, Potentiometers).
* DC Power Supplies, Digital Multimeters (DMM).
